Por que este manual foi criado?

O Slackware GNU/Linux é a distribuição mais antiga ainda em desenvolvimento, infelizmente o projeto gnu não endossa o mesmo por ter alguns software/arquivos que não respeitam a liberdade do usuário.

Existe um projeto documentado chamado freenix o qual faz uma limpeza no projeto Slackware, tornando-o assim um projeto livre mas não é adosado pelo GNU. O projeto Freenix fez toda limpeza e criou uma nova iso, porém eu (slackjeff) não vejo nenhuma necessidade de criar um novo projeto/distribuição para uma finalidade como está.

Este manual tem em mente ser o mais simples/direto e fácil possível para transformar o seu Slackware stable e current em uma distribuição 100% Livre.
